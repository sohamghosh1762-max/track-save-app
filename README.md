# Track & Save — AI Powered Finance Management Platform

Track & Save is a modern AI-powered personal finance management platform built using React, Vite, Node.js, Express.js, and MongoDB.

## Features

- Dynamic income & expense tracking
- AI-powered financial insights
- Salary & savings management
- Edit/Delete transactions
- PDF financial reports
- Recurring payments
- Interactive analytics dashboard
- MongoDB integration
- JWT authentication
- Responsive fintech UI

## Tech Stack

Frontend:
- React.js
- Vite
- Recharts

Backend:
- Node.js
- Express.js
- MongoDB Atlas

## Deployment

Frontend:
- Vercel

Backend:
- Render

Database:
- MongoDB Atlas
