Track & Save — AI Powered Finance Management Platform

Track & Save is a next-generation AI-powered personal finance management platform designed to help users track, analyze, and optimize their financial activities through an intelligent and modern fintech dashboard.

Built with a scalable full-stack architecture using React, Vite, Node.js, Express.js, and MongoDB Atlas, the platform delivers a seamless experience for managing income, expenses, savings goals, recurring payments, and financial insights in real time.

✨ Key Features
Smart income & expense tracking
AI-generated financial insights and analytics
Manual salary, savings, and transaction management
Add, edit, and delete transactions
Goal-based savings tracker
Recurring payment management
Interactive financial charts and analytics dashboard
PDF financial report generation
Secure JWT-based authentication system
MongoDB Atlas cloud database integration
Responsive premium fintech-inspired UI/UX
Dark/light theme support
Real-time dashboard updates
